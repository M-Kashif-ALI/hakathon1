# Navigation with Nav2

This chapter covers path planning and localization for bipedal humanoid robots using Navigation2 (Nav2).

## Overview

Navigation2 provides a comprehensive navigation stack for mobile robots that includes path planning, localization, and obstacle avoidance capabilities. This chapter focuses on adapting Nav2 for bipedal humanoid robots, which have unique locomotion characteristics compared to wheeled robots.

## Learning Objectives

By the end of this chapter, you should be able to:
- Understand Nav2 architecture and components
- Adapt navigation for bipedal humanoid robot characteristics
- Implement path planning for walking robots
- Configure localization for humanoid locomotion patterns
