# NVIDIA Isaac Sim

This chapter covers photorealistic simulation and synthetic data generation for perception models using NVIDIA Isaac Sim.

## Overview

NVIDIA Isaac Sim is a robotics simulation environment that provides photorealistic rendering and physics simulation capabilities. It enables developers to create realistic virtual environments for testing and training robotic systems without requiring physical hardware.

## Learning Objectives

By the end of this chapter, you should be able to:
- Understand the core capabilities of NVIDIA Isaac Sim
- Set up photorealistic simulation environments
- Generate synthetic data for perception model training
- Apply best practices for simulation-based development
