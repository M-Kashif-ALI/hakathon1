# Physical AI & Humanoid Robotics Course - Structure Information

This document provides information about the structure of the Physical AI & Humanoid Robotics Course documentation.

## Course Overview

The course is organized into multiple modules, with the first module focusing on ROS 2 as the robotic nervous system for humanoid robots.

## Module Structure

### Module 1: ROS 2 Robotic Nervous System

Located at `/docs/001-ros2-nervous-system/`, this module contains:

1. **ROS 2 Architecture** (`/docs/001-ros2-nervous-system/ros2-architecture/`) - Understanding ROS 2 as middleware for humanoid robots
2. **Python Robot Control** (`/docs/001-ros2-nervous-system/python-robot-control/`) - Implementing robot control with Python and rclpy
3. **URDF Humanoid Structure** (`/docs/001-ros2-nervous-system/urdf-humanoid-structure/`) - Defining humanoid robot structures with URDF

### Module 2: Digital Twin

Located at `/docs/digital-twin/`

### Module 3: Isaac Robot Brain

Located at `/docs/003-isaac-robot-brain/`

## Multi-Language Support

The documentation is available in four languages:
- English (default)
- Urdu
- French
- Spanish

The language switcher is available in the top navigation bar.

## Design Features

- **Animations**: Subtle CSS animations enhance the user experience on the landing page
- **Accessibility**: Full support for keyboard navigation, reduced motion preferences, and high contrast mode
- **Responsive Design**: Optimized for viewing on all device sizes
- **SEO-Friendly URLs**: Clean, descriptive URLs that represent the content hierarchy
